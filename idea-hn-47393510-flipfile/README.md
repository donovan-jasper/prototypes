# LocalForge

Privacy-first file converter suite that processes 75+ file formats entirely in your browser using WebAssembly—no uploads, no servers, just instant conversions.

## Features
- Convert between 75+ file formats (images, PDFs, media, documents, archives)
- Entirely client-side processing with WebAssembly
- No file uploads or server dependencies

## Tech Stack
- Frontend: Vanilla HTML/CSS/JavaScript
- Backend: Node.js + Express
- File Processing: WebAssembly (FFmpeg.wasm, pdf-lib, JSZip)
- Storage: Client-side only

## Running
Run `node server.js` and open http://localhost:3000 in a modern browser.
