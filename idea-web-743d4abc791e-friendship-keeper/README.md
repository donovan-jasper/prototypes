# Friendship Keeper

Never lose touch with the people who matter — smart reminders and conversation starters that keep your relationships thriving.

## What it does
• Sends automated touchpoints and conversation starters to help you stay connected with friends and family
• Tracks relationship health and suggests personalized ways to strengthen connections
• Provides shared experience features to create meaningful interactions despite busy schedules

## Monetization model
Freemium with premium relationship insights and advanced automation features

## Tech stack
React Native with Expo, Firebase backend, Node.js API

## How to run
```bash
npx expo start
```
