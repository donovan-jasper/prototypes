# BuildHiFi

> A mobile app for hobbyists to design, visualize, and optimize audio setups (like PCPartPicker but for hi-fi audio gear)

**Gap:** Existing apps focus on music streaming or basic speaker setups; none provide deep technical validation or signal chain visualization

## Features

- Broadest reach:**
- Casual upgraders** (largest segment): People replacing TV soundbars, buying first turntables, or upg
- Home theater DIYers**: Building 5.1/7.1 systems, need to match receivers to speakers without hiring 
- Vinyl newcomers**: The exploding record collector market — they need turntable/preamp/speaker chains
