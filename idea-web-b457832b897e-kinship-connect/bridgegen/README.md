# BridgeGen

Connect across generations for meaningful friendships, mentorship, and support—right from your phone.

## Setup

1. Install Node.js and npm if you haven't already.
2. Install Expo CLI globally: `npm install -g expo-cli`
3. Clone the repository: `git clone https://github.com/yourusername/bridgegen.git`
4. Navigate to the project directory: `cd bridgegen`
5. Install dependencies: `npm install`

## Running the App

### Development

1. Start the Expo development server: `npm start`
2. Scan the QR code with the Expo Go app on your iOS or Android device, or run on an emulator/simulator.

### Testing

1. Run the test suite: `npm test`
2. For coverage report: `npm test -- --coverage`
