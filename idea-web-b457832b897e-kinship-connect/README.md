# BridgeGen

A mobile app that connects people across generations for mentorship, friendship, and emotional support, leveraging location-based matching and real-time communication.

**Who it's for:** Elderly individuals (65+), young adults (18-35) without family support, and caregivers/volunteers looking to facilitate connections.

**Gap:** Existing solutions are either web-based, lack mobile-native features (e.g., push notifications, GPS tracking), or focus only on caregiving without emotional connection.

**Monetization:** Freemium model: Free basic matching, $9.99/month for premium features (video calls, group activities, priority support).

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — Requires strong vetting, safety features, and community-building to gain trust.**
