# GameTradeHub Evolution

A mobile app for gamers to trade, track, and optimize their game collections with real-time market data and instant execution, similar to stock trading platforms but for physical games.

## Features

- Real-time market data for physical games
- Instant trade execution for premium users
- Advanced analytics and collection tracking

## Monetization

Freemium with premium features: $4.99/month for instant trade execution, advanced analytics, and priority support.

## Run

```bash
npx expo start
```
