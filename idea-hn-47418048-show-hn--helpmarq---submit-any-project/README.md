# Helpmarq Evolution

A mobile app that delivers structured, actionable feedback on creative projects from real users, helping founders and designers refine their work faster.

## Features

- Structured review questions for meaningful insights
- Community feedback from real users
- Freemium model with premium templates and priority reviews

## Monetization

Freemium: Free basic feedback, premium upgrades for advanced features.

## Run

```bash
npx expo start
```
