# RaccoonAI

Unlock limitless productivity with RaccoonAI, your collaborative AI agent for anything.

## Features
- Conversational interface for natural task management
- Task chaining across unrelated domains
- Context-aware responses for accurate workflows

## Monetization
Freemium model with in-app purchases (IAP) for tiered access.

## Tech Stack
- React Native (Expo) for cross-platform development
- SQLite for local storage
- `expo-sqlite` and `react-native-vector-icons`

## Running
1. Install dependencies: `npm install`
2. Start the app: `npx expo start`
