# App Spec: Calibrate

A mobile app that gamifies decision-making confidence tracking for professionals, using probabilistic tools to improve accuracy.

## Features

- Daily decision confidence tracking
- Calibration reports and historical analysis
- Probabilistic estimation tools

## Monetization

Freemium: Free quiz + basic tracking; $4.99/month for full decision history, calibration reports, and advanced tools.

## Run

```bash
npx expo start
```
