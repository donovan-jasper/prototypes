# SkillForge

A mobile app that gamifies skill training for any mobile game's core mechanics, not just Pokémon Go, with customizable simulations and analytics

**Who it's for:** Casual and competitive mobile gamers (ages 13-35) who want to improve skills in games like Pokémon Go, Clash Royale, or mobile sports titles

**Gap:** Most skill-training apps are either game-specific or lack gamification; no platform offers cross-game training with analytics

**Monetization:** Freemium with premium features like advanced analytics ($4.99/month) and custom game templates ($9.99/year)

**Viability: 8/10 | Competition: 8/10 | Difficulty: Medium — requires game mechanics research, motion control integration, and analytics dashboards**

## Run

```bash
npx expo start
```
