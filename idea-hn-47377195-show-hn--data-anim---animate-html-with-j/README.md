# Animata

Animata is a lightweight, dependency-free animation library that enables CSS animations through HTML data attributes, simplifying the process of adding animations to web pages.

## Features

*   **Declarative animation**: Define animations using HTML data attributes.
*   **Scroll detection**: Automatically trigger animations when elements come into view using `IntersectionObserver`.
*   **Anti-FOUC (Flash of Unstyled Content) mechanism**: Ensures a smooth user experience by preventing unstyled content from being displayed.

## Tech Stack

*   **Frontend**: Vanilla HTML/CSS/JS
*   **Backend**: Node/Express (for serving static files in the demo)
