# AppVista

A mobile app for app developers to manage App Store analytics, sales, and reviews with offline access and AI-powered insights.

## Features

- Real-time and offline-capable analytics dashboard
- AI-generated insights on user reviews and trends
- Sales tracking and revenue projections

## Monetization

Freemium model with a $9.99/month tier for advanced analytics and AI features, or a one-time $49 purchase for full access.

## Run

```bash
npx expo start
```
