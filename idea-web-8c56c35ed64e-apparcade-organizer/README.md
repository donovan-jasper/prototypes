# AppArcade Organizer

> A customizable, AI-powered app launcher that intelligently organizes and launches apps based on user behavior and context, with themes and productivity workflows

**Gap:** Most launchers lack AI-driven organization or deep integration with productivity workflows

## Features

- ML/AI:** TensorFlow.js Lite for on-device pattern recognition
- Background tasks:** expo-task-manager for usage tracking
- Permissions:** expo-application, expo-device for context detection
- UI:** React Nativ
