# TaskHive

TaskHive is a dynamic task management system that adapts to your life, not the other way around. It's an AI-powered task manager that learns your rhythm and keeps you ahead, not overwhelmed.

## Features

- **AI-Powered Auto-Organization**: Tasks are auto-categorized based on keywords and past behavior.
- **Seamless Calendar Sync**: Auto-imports events from Google/Apple Calendar and sends push notifications.
- **Visual Task Board**: Drag-and-drop Kanban-style interface with color-coded priorities.
- **Location-Based Reminders**: Set reminders based on your location.
- **Cross-Device Sync**: Encrypted SQLite backup to cloud (optional).
