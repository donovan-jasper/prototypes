# PageBridge

A mobile app that syncs reading progress across digital and physical books, but broadened to any physical item with a digital counterpart (e.g., magazines, manuals, cookbooks, training materials)

**Who it's for:** Students, professionals, hobbyists, and anyone who owns both digital and physical versions of books, manuals, or training materials (e.g., engineers, chefs, DIYers)

**Gap:** No app bridges the gap between physical and digital reading progress. Users manually track pages, leading to frustration when switching devices.

**Monetization:** Freemium with a $4.99/year premium tier for advanced features (OCR, manual sync, cloud backup). Ads for free users.

**Viability: 7/10 | Competition: 8/10 | Difficulty: Medium — requires robust OCR and cloud sync, but no major technical hurdles beyond existing tech.**

## Run

```bash
npx expo start
```
