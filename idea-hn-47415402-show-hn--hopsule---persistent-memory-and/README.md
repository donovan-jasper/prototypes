# Hopsule → Architect

A mobile app that enforces architectural consistency in AI-generated code by storing and injecting team-specific rules into AI coding tools.

## Features
- Store and manage team-specific architectural rules
- Inject rules into AI coding assistants (Cursor, Copilot, Claude)
- Collaborate with team members in real-time

## Monetization
Team-based SaaS model ($10-$20/user/month) with free tier for individuals.

## Run
```bash
npx expo start
```
