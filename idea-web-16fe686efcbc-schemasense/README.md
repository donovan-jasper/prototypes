# QuerySnap

An AI-powered database assistant app that helps non-technical users generate SQL queries, documentation, and reports from their database schemas on mobile devices.

**Who it's for:** Business analysts, small business owners, and field service technicians who need to interact with databases without writing SQL.

**Gap:** Most mobile database tools are either too technical or lack AI-powered assistance for non-developers.

**Monetization:** Freemium model with a $9.99/month premium tier for advanced AI features and offline access.

**Viability: 7/10 | Competition: 9/10 | Difficulty: Medium — requires strong AI integration and database connectivity, but mobile-first design is feasible.**

## Run

```bash
npx expo start
```
