# QuerySnap

On-the-go database insights without writing SQL

## Features
- Natural language queries ("Show me top customers") with AI-powered SQL generation
- Offline-first design with cached query patterns
- Voice-to-query interface for hands-free operation

## Monetization
Freemium model with free tier (10 queries/month) and premium ($9.99/month) for unlimited access

## Tech Stack
- React Native (Expo)
- SQLite (expo-sqlite)
- OpenAI API (GPT-4)
- Zustand (state management)
- React Native Paper (UI components)

## Running
```bash
npx expo start
```
