# CramCaster Evolution: FocusFlow

A distraction-free, calendar-integrated productivity app for professionals and students who struggle with focus and time management.

## Features

- Distraction-free study/work sessions
- AI-powered task prioritization (Premium)
- Calendar integration for seamless scheduling

## Monetization

Freemium with $4.99/month premium tier for advanced features.

## Run

```bash
npx expo start
```
