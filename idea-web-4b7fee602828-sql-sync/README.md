# SQL Sync

> A mobile app that helps non-technical users create and manage simple databases (like spreadsheets) with AI-powered documentation and syncing

**Gap:** Existing tools require technical knowledge or are too rigid for non-developers; no seamless AI documentation for mobile users

## Features

- Parents** organizing kids' activities, medical records, and school contacts
- Hobbyists** cataloging collections (books, plants, recipes, gear)
- Freelancers** tracking clients, invoices, and project notes without accounting software
- Event planners** managing guest lists, vendors, and budgets
