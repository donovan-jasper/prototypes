# DataPal

A mobile app that helps non-technical users create and manage simple databases (like spreadsheets) with AI-powered documentation and syncing

**Who it's for:** Small business owners, educators, and hobbyists who need structured data but lack SQL expertise

**Gap:** Existing tools require technical knowledge or are too rigid for non-developers; no seamless AI documentation for mobile users

**Monetization:** Freemium model with $5/month for advanced AI features and cloud sync; one-time $20 for premium templates

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires UX for non-technical users and AI integration**

## Run

```bash
npx expo start
```
