# PosturePal

Turn your phone into a personal posture coach that fixes your neck pain in 5 minutes a day.

## Features
- 5-minute daily posture exercises with motion detection
- Pain tracking and progress analytics
- Weekly posture photo comparison
- Smart reminders based on your screen time

## Monetization
Freemium model with core features free, premium unlocking advanced exercises and analytics.

## Tech Stack
- React Native (Expo)
- TypeScript
- SQLite (local storage)
- expo-sensors (motion detection)
- Zustand (state management)

## Running
```bash
npx expo start
```
