# SpontaPlay Mobile App

## Setup Instructions

1. Clone the repository: `git clone https://github.com/yourusername/spontaplay.git`
2. Navigate to the project directory: `cd spontaplay`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

## Run Instructions

1. Ensure you have the Expo Go app installed on your mobile device.
2. Scan the QR code displayed in the terminal after running `npm start`.
3. The app should now be running on your mobile device.
