# SleepyBook → DriftGuard

A sleep-focused productivity app that uses phone sensors (motion, sound, ambient light) to detect sleep onset and automatically adjusts content (e.g., work tasks, learning modules) to optimize focus before sleep.

**Who it's for:** Remote workers, students, and professionals who struggle with sleep disruption during work hours. Also, parents of young children who need to monitor sleep patterns.

**Gap:** Existing sleep apps focus on tracking or ambient sounds; none dynamically adjust content based on real-time sleep detection. This fills a gap for people who need to finish tasks before sleep.

**Monetization:** Freemium model with premium features like advanced sleep analytics ($2.99/month) and integrations with productivity tools (e.g., Trello, Notion).

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires sensor calibration and UX design for real-time adjustments without user frustration.**
