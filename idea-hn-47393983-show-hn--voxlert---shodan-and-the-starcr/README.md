# VoiceVerse - AI Narrator for Your Digital Life

Personalized AI narrator system that provides custom voice feedback for any app activity, task completion, or notification using character voices and contextual messaging for gamers and productivity enthusiasts.

## Features

- Custom character voices for app notifications and activities
- Contextual messaging based on task completion and app usage
- Cross-platform voice feedback for multiple simultaneous applications

## Monetization

Freemium model - free tier with 5 characters and basic voices, premium $4.99/month for unlimited characters.

## Run

```bash
npx expo start
```
