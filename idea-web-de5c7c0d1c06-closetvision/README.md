# ClosetVision Evolution

An AI-powered personal stylist that suggests outfits, offers virtual try-ons, and provides style advice for any occasion.

## Features

- AI-powered outfit suggestions using phone camera and wardrobe data
- Virtual try-on for real-time fashion visualization
- Personalized style advice for any occasion

## Monetization

Freemium model with premium features at $4.99/month.

## Run

```bash
npx expo start
```
