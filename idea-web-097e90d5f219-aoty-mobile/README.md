# CritWave

A personalized, notification-driven music discovery app that aggregates album reviews, tracks trends, and surfaces critical consensus for music fans and collectors

**Who it's for:** Music enthusiasts (ages 18-45), indie/vinyl collectors, and critics who want real-time album updates and consensus tracking

**Gap:** Existing apps lack real-time notifications for followed artists or a unified 'Album of the Year' consensus tracker

**Monetization:** Freemium with a $4.99/month premium tier for ad-free access, exclusive artist interviews, and early release previews

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires data aggregation, real-time notifications, and music API integrations**

## Run

```bash
npx expo start
```
