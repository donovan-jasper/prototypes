## Features

- **Quick Send Flow**: Search for recipient, choose restaurant, add message, schedule delivery
- **Gift Tracking**: Real-time delivery status with map view
- **Gift History**: View past gifts and use "Send Again"
- **Social Sharing**: Share gifts on social media and see community feed
- **Subscription**: Upgrade to GiftGrub Plus for unlimited gifts and premium features

## Tech Stack

- **Framework**: React Native (Expo)
- **Navigation**: Expo Router
- **State Management**: Zustand
- **Local Storage**: SQLite (expo-sqlite)
- **API Integration**: REST APIs (mock initially)
- **Push Notifications**: Expo Notifications
- **Payments**: Stripe SDK for React Native
- **Maps**: React Native Maps
- **Testing**: Jest + React Native Testing Library
