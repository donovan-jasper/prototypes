# CareQuest

A habit-building app that gamifies preventive healthcare tracking for families, with social accountability and rewards for completing checkups

**Who it's for:** Parents of young children (ages 0-12), busy professionals (ages 30-50), and seniors (ages 65+) who need to coordinate care for multiple family members

**Gap:** Existing apps focus on individual tracking; this app lacks social features, gamification, and family coordination tools

**Monetization:** Freemium with premium features like provider discounts ($5/month) and family group analytics ($10/month)

**Viability: 7/10 | Competition: 4/10 | Difficulty: Medium — requires healthcare data partnerships and UX for complex family workflows**

## Run

```bash
npx expo start
```
