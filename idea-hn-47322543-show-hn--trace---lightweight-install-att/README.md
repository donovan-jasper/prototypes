# Trace

One sentence describing what the app does and who it's for: Trace is a mobile app that helps mobile app marketers, growth hackers, and app developers track and optimize their app install sources and deep links for better user acquisition and engagement.

## Features

- Lightweight install attribution tracking
- Deep link management for seamless user experience
- Advanced analytics for data-driven decision making

## Monetization

One line: Freemium model with a $29/month premium tier for advanced analytics and automation features.

## Run

```bash
npx expo start
```
