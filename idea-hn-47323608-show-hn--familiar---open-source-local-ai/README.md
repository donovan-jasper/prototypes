# NightOwl AI

Your phone works while you sleep — wake up to organized files, summarized documents, and completed tasks, all processed locally without the cloud.

## Features
- Night Shift Scheduler: Auto-process files during charging/idle periods
- Smart File Organizer: Categorize photos, documents, and receipts
- Document Processor: OCR, summarization, and key info extraction
