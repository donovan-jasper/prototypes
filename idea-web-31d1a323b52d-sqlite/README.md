# DataPal

An AI-powered database assistant that helps non-technical users create, manage, and query databases on mobile without writing SQL

**Who it's for:** Business owners, marketers, and analysts who need databases but lack technical skills; also solo developers who want faster prototyping

**Gap:** Most mobile database tools require coding knowledge or are too simplistic; none combine AI-powered query generation with offline database creation

**Monetization:** Freemium model with $5/month for advanced features like multi-table relationships and GPT-powered query suggestions

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires strong AI integration and intuitive UI for non-technical users**

## Run

```bash
npx expo start
```
