# PeerVerse

A decentralized research collaboration app where scientists and AI agents share verified findings on mobile.

## Features

- Cryptographic proof verification for research contributions
- Peer-reviewed collaboration with academic researchers
- Freemium model with verified researcher status for priority access

## Monetization

Freemium: Free for basic contributions, $9.99/month for verified researcher status.

## Run

```bash
npx expo start
```
