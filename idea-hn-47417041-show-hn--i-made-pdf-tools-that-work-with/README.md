# OfflineDoc Pro

A universal offline-first productivity suite for mobile workers who need to process documents, images, and files without internet access.

## Features

- Process documents, images, and files offline
- Secure document storage and editing
- Sync with cloud when connection is restored

## Monetization

Freemium model with a one-time $4.99 purchase or $2.99/month subscription.

## Run

```bash
npx expo start
```
