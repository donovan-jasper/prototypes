# CodeCapsule

## One-line pitch
Run AI coding experiments in bulletproof sandboxes on your phone—no setup, no risk, just code.

## Expanded vision

### Who is this REALLY for?

**Primary audience:** Mobile-first developers (students, freelancers, indie hackers) who want to code on the go without carrying a laptop. They use AI assistants but fear breaking their device or exposing sensitive data.

**Broader audience:**
- **Learning developers** who want to experiment with AI-generated code snippets without polluting their main environment
- **Tech-curious non-developers** who want to try coding with AI guidance in a safe playground (think "Duolingo for coding with AI")
- **Security-conscious professionals** who need to test untrusted code samples from Stack Overflow, GitHub, or AI assistants
- **Educators and bootcamp instructors** who want students to have isolated environments for assignments
- **Content creators** (YouTubers, bloggers) who demo code and need quick, disposable environments

### Adjacent use cases:
- **API testing playground** — test third-party APIs without exposing credentials
- **Code snippet validator** — paste code from the internet and run it safely before adding to your project
- **Learning sandbox** — follow coding tutorials with AI assistance in isolated environments
- **Interview prep
