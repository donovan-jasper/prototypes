# GitHub AI

A mobile app that automates GitHub workflows with AI agents for developers on the go.

## Features

- AI-powered Scrum Master for issue tracking
- Automated code reviews and testing
- Seamless GitHub integration for mobile

## Monetization

Freemium: Free for basic issue tracking, $9/month for AI agent automation.

## Run

```bash
npx expo start
```
