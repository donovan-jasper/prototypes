# StoryForge AI

Turn any story, script, or idea into professional animated videos using AI, designed for content creators, educators, and small business owners.

## Features

- AI-powered animation generation from text stories and scripts
- Full creative control over video output
- No ongoing subscription required for basic use

## Monetization

Freemium model - free basic videos up to 30 seconds, $9.99/month for unlimited length and advanced features.

## Run

```bash
npx expo start
```
