# EquityEdge

A mobile app that aggregates and simplifies startup equity liquidity options, providing side-by-side comparisons and demand aggregation to negotiate better terms for employees.

**Who it's for:** Recent or soon-to-be laid-off software engineers, startup employees with equity, and financial advisors helping clients with startup exits.

**Gap:** No mobile app exists to compare liquidity providers, track equity value in real-time, or build collective demand to negotiate better terms.

**Monetization:** Freemium model with a $9.99/month premium tier for advanced scenario modeling, provider alerts, and waitlist priority access.

**Viability: 7/10 | Competition: 10/10 | Difficulty: Medium—requires financial data integration, regulatory compliance, and liquidity provider partnerships.**
