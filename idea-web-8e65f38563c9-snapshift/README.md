# VoiceVault

A mobile app that delivers personalized, on-demand motivational coaching for daily goals, leveraging voice prompts and reminders to boost productivity and habit formation.

**Who it's for:** Busy professionals, students, and parents who struggle with focus, procrastination, or habit formation, and would pay for a structured motivational system.

**Gap:** Most apps focus on task management or gamification; few combine voice motivation with real-time reminders in a seamless, subscription-based model.

**Monetization:** Subscription model ($5-$10/month) for unlimited voice prompts and personalized coaching, with a freemium tier offering basic reminders.

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires strong voice content creation, user engagement strategies, and monetization testing.**

## Run

```bash
npx expo start
```
