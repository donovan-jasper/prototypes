# MindMeld

A personalized learning app that adapts to your pace for any skill, using spaced repetition and mastery tracking to reduce stress and maximize retention.

## Features

- Adaptive spaced repetition for any skill
- Mastery tracking and progress analytics
- Offline access and ad-free experience (premium)

## Monetization

Freemium with premium features starting at $4.99/month.

## Run

```bash
npx expo start
```
