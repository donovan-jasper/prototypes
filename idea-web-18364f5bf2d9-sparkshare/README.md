# IdeaSpark

Turn fleeting ideas into real projects with instant feedback from people who've been there.

## Features
- Swipeable feed of ideas with voting and discussion
- Voice-to-text idea submission with auto-categorization
- Real-time feedback notifications and threaded discussions
- Gamified reputation system with badges and leaderboards
