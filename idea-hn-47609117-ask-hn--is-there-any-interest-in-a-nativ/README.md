# DiscordX

A fast, native Discord client for mobile with offline support, optimized for power users and casual users alike.

## Features

- Lightning-fast performance with native UI
- Full offline messaging and media caching
- Custom themes and premium emotes

## Monetization

Freemium with optional Nitro features via in-app purchases.

## Run

```bash
npx expo start
```
