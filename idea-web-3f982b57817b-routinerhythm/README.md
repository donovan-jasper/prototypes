# FlexFlow - Your Life, Automatically Organized

FlexFlow is a mobile app that automatically organizes your life around your ever-changing schedule, so you never miss what matters. Built for shift workers, students, parents, and anyone with an unpredictable schedule.

## Features

- **Dynamic Schedule Input**: Quick-add work shifts, appointments, and commitments
- **Adaptive Task Queue**: Tasks automatically suggest themselves in available time slots
- **Flexible Routines**: Routines that adapt to your schedule (morning routine adjusts for early/late shifts)
- **Proactive Notifications**: Context-aware reminders based on your availability
- **Energy-Aware Scheduling** (Premium): Learn your productive hours and optimize scheduling

## Tech Stack

- React Native (Expo SDK 52+)
- SQLite for local-first data
- Zustand for state management
- date-fns for date/time operations
- expo-notifications for proactive alerts
- React Native Paper for UI components
