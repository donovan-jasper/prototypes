# FinFlow

Your money, unified — track spending, investments, and net worth in one beautiful, privacy-first dashboard.

## Features
- Unified dashboard showing net worth, spending trends, and investment performance
- Quick transaction entry with optional receipt scanning
- Portfolio tracking with manual asset management and price updates

## Monetization
Free tier with basic features; premium unlocks unlimited categories, real-time updates, and cloud backup.

## Tech Stack
- React Native (Expo)
- SQLite for local storage
- react-native-chart-kit for visualizations
- TypeScript for type safety

## Running
```bash
npx expo start
```
