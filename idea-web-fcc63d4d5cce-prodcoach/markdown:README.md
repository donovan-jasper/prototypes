# MotiMate - Your Pocket Coach

## Overview
MotiMate is an AI-powered mobile application that transforms everyday tasks into wins through encouragement and smart habit tracking. It's designed for people who want to make the most of their frequent phone interactions by turning them into productive moments.

## Setup Instructions

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Expo CLI (for mobile development)

### Installation Steps
1. Clone this repository
2. Navigate to the project directory
3. Run `npm install` or `yarn install` to install dependencies
4. Run `npx expo install` to install Expo dependencies
5. Start the development server with `npx expo start`
