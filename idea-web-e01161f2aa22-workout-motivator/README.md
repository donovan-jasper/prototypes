# Workout Motivator

> A mobile app that provides real-time motivational coaching for any task or activity, not just workouts, using voice prompts and personalized encouragement.

**Gap:** Most apps focus on workouts or productivity separately. Few combine voice coaching with gamification for any task, and none leverage ambient audio cue

## Features

- ADHD/neurodivergent users** who need external accountability and dopamine hits to complete tasks
- Remote workers** drowning in Zoom fatigue who need ambient motivation during deep work
- Parents** trying to make chores fun for kids (or themselves)
- Language learners** who need encouragement during practice sessions
