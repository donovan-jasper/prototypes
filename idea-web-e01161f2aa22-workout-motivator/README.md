# MotiveMate

A mobile app that provides real-time motivational coaching for any task or activity, not just workouts, using voice prompts and personalized encouragement.

**Who it's for:** Busy professionals, fitness enthusiasts, students, and anyone who struggles with motivation during repetitive tasks (e.g., studying, chores, commuting).

**Gap:** Most apps focus on workouts or productivity separately. Few combine voice coaching with gamification for any task, and none leverage ambient audio cues as effectively.

**Monetization:** Freemium with premium voice coaches ($4.99/month), customizable prompts ($2.99/week), and a 'Motivation Boost' subscription ($9.99/month for unlimited prompts).

**Viability: 7/10 | Competition: 6/10 | Difficulty: Medium — requires voice synthesis, gamification design, and task tracking, but no major technical hurdles.**

## Run

```bash
npx expo start
```
