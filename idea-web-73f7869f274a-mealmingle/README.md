# MealMingle Spec

A social meal-sharing app that lets users coordinate group orders, split costs, and manage shared deliveries for events, work lunches, or casual gatherings

**Who it's for:** Young professionals (25-40), event planners, and families who frequently order food together but struggle with coordination and cost-splitting

**Gap:** Existing apps don't natively integrate group ordering with delivery tracking and cost-splitting in one seamless flow

**Monetization:** Freemium model with $5/month for premium features like automated cost-splitting, priority delivery, and group templates

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires smooth UX for group coordination and payment integration**

## Run

```bash
npx expo start
```
