# NearNow - Meet people nearby who want to do what you want to do, right now

## Overview
NearNow is a mobile application that connects people in real-time based on location and shared activities. Users can broadcast what they want to do right now, find others nearby doing the same activity, and connect instantly.

## Tech Stack
- React Native (Expo SDK 52+)
- Expo Router (file-based routing)
- Zustand (state management)
- Expo SQLite (local storage)
- expo-location (geolocation)
- expo-notifications (push notifications)
- Supabase (backend, auth, real-time)
- react-native-maps (map visualization)
- Jest + React Native Testing Library (testing)

## Prerequisites
- Node.js (v16 or higher)
- Expo CLI (`npm install -g @expo/cli`)
- iOS Simulator or Android Emulator (for development)
- Physical device recommended for location/testing
