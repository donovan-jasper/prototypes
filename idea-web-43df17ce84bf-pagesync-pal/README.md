# PageSync Pal

> A universal progress-tracking app for any book, movie, or media that syncs across formats and devices

**Gap:** No single app unifies progress tracking across ALL formats (books, movies, audiobooks) and devices (phone, tablet, laptop)

## Features

- A universal progress-tracking app for any book, movie, or media that syncs across formats and device
- Phone-native advantage: instant access to sync progress via camera (barcode/QR code scanning), voice
