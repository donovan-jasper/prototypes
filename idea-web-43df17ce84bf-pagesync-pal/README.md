# PageFlow

A universal progress-tracking app for any book, movie, or media that syncs across formats and devices

**Who it's for:** Students, professionals, and casual readers who own both physical and digital copies of books, movies, or audiobooks; frequent travelers or commuters who switch devices often

**Gap:** No single app unifies progress tracking across ALL formats (books, movies, audiobooks) and devices (phone, tablet, laptop)

**Monetization:** Freemium with premium features like offline sync, advanced barcode scanning, and custom progress sharing; one-time $4.99 for lifetime access

**Viability: 8/10 | Competition: 8/10 | Difficulty: Medium — requires robust barcode/QR scanning, API integrations, and cross-platform syncing**

## Run

```bash
npx expo start
```
