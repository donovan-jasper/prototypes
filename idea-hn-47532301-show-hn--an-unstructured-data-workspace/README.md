# DataSift

A mobile-first AI tool that helps non-technical users extract insights from unstructured data with SQL-like filtering and cost optimization.

## Features

- AI-powered analysis of text, audio, and files
- SQL-like filtering for structured insights
- Cost optimization for large datasets

## Monetization

Freemium: Free for basic LLM operations (100 files/month), $9.99/month for unlimited pro.

## Run

```bash
npx expo start
```
