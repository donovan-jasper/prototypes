# Octopoddy Evolution

An AI-powered podcast assistant that enhances listening with ad skipping, chapter navigation, and smart controls for mobile users seeking seamless, ad-free experiences.

## Features

- AI-powered ad skipping and chapter navigation
- Smart playback controls for effortless listening
- Premium tier with AI-generated chapter summaries

## Monetization

Freemium model with a $4.99/month premium tier for advanced features.

## Run

```bash
npx expo start
```
