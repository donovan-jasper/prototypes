# Memora - AI Photo Restoration App

## Overview
Memora is a mobile app that uses AI to restore, enhance, and relive your memories by transforming old, damaged photos into stunning, shareable masterpieces.

## Features
- **AI Restoration Hub**: One-tap restoration for scratches, blurs, and colorization.
- **Smart Enhancer**: Auto-adjusts brightness, contrast, and sharpness.
- **Artistic Styles**: Apply vintage, modern, or custom AI-driven filters.
- **Batch Processing**: Restore multiple photos at once (paid feature).
- **Cloud Backup**: Secure storage for all restored photos (paid feature).

## Tech Stack
- **Frontend**: React Native (Expo)
- **Local storage**: SQLite
- **AI processing**: Firebase ML Kit or custom TensorFlow Lite
- **Backend**: Firebase (Auth, Firestore, Functions)
- **Dependencies**: React Navigation, Redux, Expo ImagePicker
