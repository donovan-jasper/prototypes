# HobbyHub

A hyper-local social app that connects people for shared interests (beyond board games) through real-time meetups and community-building features

**Who it's for:** Adults aged 18-45 who enjoy social hobbies, including gamers, fitness enthusiasts, book clubs, and hobbyists

**Gap:** Most apps focus on large-scale events; this app prioritizes small, organic, in-person meetups with strong local community features

**Monetization:** Freemium with premium features like ad-free browsing, verified profiles, and exclusive event hosting; sponsorships from local businesses

**Viability: 7/10 | Competition: 5/10 | Difficulty: Medium — requires strong local partnerships and community moderation**

## Run

```bash
npx expo start
```
