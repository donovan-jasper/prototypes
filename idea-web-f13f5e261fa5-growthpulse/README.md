# GrowthPulse

A mobile-first habit and progress tracker that gamifies personal growth by syncing health, productivity, and mood data for busy professionals (25-45).

## Features
- Syncs with health, productivity, and mood data
- AI-driven insights and social accountability
- Gamified progress tracking

## Monetization
Freemium model with a $9.99/month premium tier for advanced analytics and social features.

## Run
```bash
npx expo start
```
