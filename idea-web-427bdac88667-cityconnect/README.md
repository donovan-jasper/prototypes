# LocalLoop

A hyper-local community app that helps people discover and organize spontaneous social activities, events, and meetups based on their interests and location

**Who it's for:** Young professionals (25-40), parents with kids, hobbyists, and anyone seeking low-pressure social connections in their area

**Gap:** Existing apps are either too formal (Meetup), too broad (Facebook Events), or lack real-time local discovery. CityConnect could focus on micro-activities (e.g., 'who wants to play chess tonight?') and hyper-local networking

**Monetization:** Freemium with premium features like ad-free browsing, exclusive event hosting, or a 'verified organizer' badge for $5/month

**Viability: 7/10 | Competition: 7/10 | Difficulty: Medium — requires strong community engagement and local partnerships to stand out**

## Run

```bash
npx expo start
```
