# BookNest

A premium, one-time-purchase ebook reader app that simplifies digital library management and eliminates subscription hassles for serious readers.

**Who it's for:** Tech-savvy readers, students, professionals, and hobbyists who own large ebook collections and dislike subscription-based readers.

**Gap:** Existing readers lack a one-time purchase model with robust file management and no forced cloud sync. Many require constant internet or have bloated interfaces.

**Monetization:** One-time purchase of $29.99 (or $49.99 for advanced features) to avoid subscription fees and recurring charges.

**Viability: 7/10 | Competition: 6/10 | Difficulty: Medium — requires strong file management and format support, but the niche audience justifies it.**

## Run

```bash
npx expo start
```
