# CodeShift

A mobile app that helps mid-career engineers pivot into AI/embedded systems with curated learning paths and project templates.

## Features

- Curated learning paths tailored to existing skills
- Project templates for hands-on practice
- Community collaboration tools for peer support

## Monetization

Freemium model: Free access to basic features, premium subscription at $15/month.

## Run

```bash
npx expo start
```
