# GearGuru

Your game's second brain—manage inventories, optimize builds, and dominate the meta across all your favorite RPGs and loot-shooters.

## Features
- Universal inventory sync across multiple games
- Auto-optimized builds with stat weights
- Push alerts for in-game vendor resets

## Monetization
Freemium model with premium features like unlimited game integrations and advanced analytics.

## Tech Stack
- React Native (Expo)
- SQLite for offline storage
- Firebase Auth & FCM
- RESTful APIs (with web scraping fallback)

## Running
`npm install && expo start`
