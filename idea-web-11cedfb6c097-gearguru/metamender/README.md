# MetaMender

## Overview
MetaMender is a cross-platform app designed to help gamers manage their inventories, optimize builds, and stay ahead of the meta across multiple RPGs and loot-shooters.

## Features
- **Universal Inventory Manager**: Sync and organize gear across multiple games.
- **Build Optimizer**: Auto-generate and compare builds with stat weights.
- **Dynamic Vendor Alerts**: Push notifications for in-game vendor resets.
- **Meta Tracker**: Real-time trends for gear/weapons.
- **Community Builds**: Share/import builds from a trusted database.

## Tech Stack
- **Frontend**: React Native (Expo)
- **Local Storage**: SQLite
- **APIs**: RESTful endpoints
- **Auth**: Firebase Auth
- **Push Notifications**: Firebase Cloud Messaging (FCM)
