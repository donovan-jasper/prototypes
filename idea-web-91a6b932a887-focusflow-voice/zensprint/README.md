# ZenSprint

Your AI focus coach that turns work sessions into rewarding sprints—with voice guidance, streak tracking, and friends who keep you accountable.

## Features

- **Voice-guided focus sessions** with AI coaching
- **Streak & reward system** with level progression
- **Accountability pods** to stay motivated with friends
- **Smart reminders** to help build consistency
- **Session analytics** to track your productivity

## Tech Stack

- React Native (Expo SDK 52+)
- Expo AV (background audio)
- Expo Speech (text-to-speech)
- Expo Notifications (push reminders)
- SQLite (local data storage)
- Zustand (state management)
- date-fns (date utilities)
