# Verdant — Universal Plant Care Companion

## Setup Instructions

1. Clone the repository: `git clone https://github.com/yourusername/verdant.git`
2. Navigate to the project directory: `cd verdant`
3. Install dependencies: `npm install`
4. Set up the database: `npm run setup-db`
5. Start the development server: `npm start`

## Run Instructions

1. Start the development server: `npm start`
2. Open your browser and navigate to `http://localhost:3000`

## Features

- Track every plant's journey from seedling to thriving greenery
- Photo timelines to document growth
- Smart care reminders to prevent neglect
- Community features to share green thumb wins
