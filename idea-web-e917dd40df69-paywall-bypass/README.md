# Paywall Bypass

A mobile app that unlocks premium content across news, magazines, and streaming platforms with smart discovery—built for students, professionals, and readers who can't afford multiple subscriptions.

## Features

- Unified interface to access premium content across all major platforms
- Smart content discovery that learns your reading preferences
- Ad-free experience with priority content access (premium tier)

## Monetization

Freemium model with $2.99/month premium tier for ad-free access and priority content.

## Run

```bash
npx expo start
```
