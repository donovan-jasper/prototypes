# SkillHive

A mobile-first habit tracker for lifelong learning, with gamified progress and social accountability for professionals and students.

## Features

- Gamified habit tracking with streaks and rewards
- Social accountability with group challenges
- Advanced analytics for progress insights

## Monetization

Freemium: Free core tracking; $9.99/month for advanced analytics, badges, and group challenges.

## Run

```bash
npx expo start
```
